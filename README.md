# Luxe Cars – Premium Automotive Showcase

A high-fidelity, world-class front-end prototype for a luxury pre-owned car dealership. Built to mirror the digital showrooms of brands like Rolls Royce and Lamborghini. 

This prototype emphasizes cinematic visuals, buttery-smooth physics, and zero-friction conversion paths to build trust with high-net-worth buyers.

## 💎 Key Features
- **Cinematic Interactions:** Custom GSAP scroll animations, magnetic sliding button hovers, and a cinematic letterbox header.
- **Inertia Scrolling:** Uses Lenis for ultra-premium, heavy momentum scrolling physics.
- **Dynamic Content:** IntersectionObserver-powered smooth number count-ups and infinite-scrolling brand marquees.
- **Frictionless Conversions:** Direct-scroll "Enquire Now" links pre-fill the contact form with the specific vehicle details.
- **Responsive Architecture:** Desktop features a custom tracking cursor, while mobile adapts perfectly into an app-like sticky bottom navigation bar.

## 🛠 Tech Stack
- **HTML5 & CSS3** (Vanilla, zero UI libraries or frameworks)
- **JavaScript** (Vanilla)
- **GSAP & ScrollTrigger** (for cinematic entrance animations)
- **Lenis** (for smooth scroll physics)
- **Swiper.js** (for swipeable inventory carousel)

## 🚀 Running Locally
Simply serve the directory with any local HTTP server:
```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080` in your browser.
